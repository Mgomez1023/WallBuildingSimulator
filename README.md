# UPS Wall Builder

A browser-based package wall-building game built with React, TypeScript, Vite, Phaser, and Matter physics.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

The repository includes `vercel.json`, so Vercel will use:

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

No environment variables are required.

Deploy by importing the repository into Vercel, or from this directory with:

```bash
npx vercel
```
